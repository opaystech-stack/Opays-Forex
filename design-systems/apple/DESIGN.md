# OpaysFox (Apple Design System Edition)

This is the customized **Apple Design System** configuration for **OpaysFox**, a premium React + Vite PWA for forex/treasury tracking with Supabase-backed data.

- **Logo**: `/logo_fox-icone.png`
- **Description**: OpaysFox is a real-time multi-currency treasury and cashbox tracking application for forex kiosks, using Google Maps mobile ergonomics.

---

## 1. Brand Identity & Overview
*   **Name**: OpaysFox
*   **Core Utility**: Multi-currency ledger, live rates converter, expense tracker, debt manager, and WhatsApp confirmation drafts.
*   **Design Paradigm**: Clean iOS/macOS human interface guidelines with system colors, rounded corners, and absolute layout clarity.

---

## 2. Visual Tokens (Apple Inspired)

### A. Colors
*   `--primary`: `#007AFF` (Apple System Blue - Brand Accent)
*   `--bg-light`: `#F2F2F7` (Apple System Light Gray Background)
*   `--card-bg`: `#FFFFFF` (Pure white card surfaces)
*   `--border-color`: `#D1D1D6` (System Gray 4 for separator borders)
*   `--text-primary`: `#000000` (Default label color)
*   `--text-secondary`: `#8E8E93` (Secondary label color)

### B. Typography
*   **Font**: System Font (`-apple-system`, `BlinkMacSystemFont`, `San Francisco`)
*   **Headings**: Bold, prominent with letter-spacing tracking.
*   **Body**: Highly readable SF Pro text.

### C. Shapes & Borders
*   **Card Radius**: `12px` / `16px` (Apple standard corner radius)
*   **Borders**: `0.5px solid var(--border-color)` (Hairline thin borders)
*   **Shadows**: Subtle elevation shadow.
